Domain name resolution change notification

April 4 , 2018

Honorable user:

Target will make the following changes to the IP segment of internal and external domain names from April 17, 2018. The expected completion time is three weeks:

1, Eastern 1 (Virginia) external domain name:

The DNS record of oss-us-east-1.aliyuncs.com and *.oss-us-east-1.aliyuncs.com is parsed to 47.252.0.0/16 segment.

2, Eastern 1 (Virginia) intranet domain name:

The DNS record of oss-us-east-1-internal.aliyuncs.com and *.oss-us-east-1-internal.aliyuncs.com is parsed to 100.115.0.0/16 segment.Users who use firewall or security group security mechanisms need to check:

1. If there is a configuration ACL or security group in the machine that can not access the IP segment after the above changes,the 47.252.0.0/16 and 100.118.0.0/16 segments can be added to the network ACL before March 23rd

2. Whether or not directly using IP or directly configuring OSS, network domain name and IP parsing mapping in the machine,and not accessing OSS by using the internal and external domain name.
